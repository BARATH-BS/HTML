HTML

HTML stands for Hypertext Markup Language

HTML is the most basic building block of the Web.

HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

This is thead tag

General Syntax

<thead>
<th> Column 1 name </th>
<th> Column 2 name </th>
</thead>


  • thead tag represents the column headings.

  • thead tag consists of all column names.

  • th tags are used to create columns

Note: thead is not necessory, thead is just used to give a column names.

This is tr tag

General Syntax

<tr>

<td> Content for column 2 </td>
</tr>


  • tr tag represents the row values.

  • tr tag consists of all column values.

This is rowspan attribute

General Syntax

</tr>


  • tr tag represents the row values.

  • rowspan attribute specifies the number of rows a cell should span.

  • n represents number of cells

</tr>


  • tr tag represents the row values.

SYMBOLS AND EMOJIS

  • Few characters and symbols requires special way of representing them due to thier use in syntax of a language.

  • Many mathematical, technical, and currency symbols, are not present on a normal keyboard.

  • To add such symbols to an HTML page, you can use the entity name or the entity number (a decimal or a hexadecimal reference) for the symbol.

  • Emojis are characters from the UTF-8 character set.

  • Emojis look like images, or icons, but they are not. They are letters (characters) from the UTF-8 (Unicode) character set.

  • UTF-8 is the default character set in HTML
